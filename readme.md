# Personal Portfolio Website

A modern and responsive personal portfolio website built to showcase my projects, skills, experience, and achievements.

## Features

* Responsive design for all devices
* Interactive UI and animations
* Project showcase section
* Skills and experience section
* Contact form integration using EmailJS
* Resume download functionality

## Tech Stack

* Next.js
* React.js
* JavaScript
* Tailwind CSS
* EmailJS

## Installation

```bash
git clone https://github.com/Neshabansari/Personal_Website_02.git
cd Personal_Website_02
npm install
npm run dev
```
